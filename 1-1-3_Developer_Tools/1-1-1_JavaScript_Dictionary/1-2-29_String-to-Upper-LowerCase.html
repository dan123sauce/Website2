String to Upper/Lower Case


Defination:

* The toLowerCase() method converts a string to lowercase letters does not change the original string.

Syntax:

* string.toLowerCase()

* string.toUpperCase()


Example:

let text = "Hello World!";

- Convert to Lower Case:

let result = text.toLowerCase();

-------------------------------------------------------------------------------------------------------------- --------------------------------------------------------------------------------------------------------------

let text = "Hello World!";

- Convert to Upper Case:

let result = text.toUpperCase();
