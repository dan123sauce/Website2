JavaScript parseInt()


Defination:

* The parseInt method parses a value as a string and returns the first integer.

* A radix parameter specifies the number system to use:

* 2 = binary, 8 = octal, 10 = decimal, 16 = hexadecimal.

* If radix is omitted, JavaScript assumes radix 10. If the value begins with "0x", JavaScript assumes radix 16.


Syntax:

parseInt(string, radix)

--- value = The value to be parsed.

--- radix = A number (2 to 36) specifying the number system.


Examples:

parseInt("10");

Output = 10

parseInt("10.00");

Output = 10

parseInt("10.33");

Output = 10

parseInt("34 45 66");

Output = 34

parseInt(" 60 ");

Output = 60

parseInt("40 years");

Output = 40

parseInt("He was 40");

Output = NaN -------------------------------------------------------------------------------------------------------------- --------------------------------------------------------------------------------------------------------------

parseInt("10", 10);;

Output = 10

parseInt("010");;

Output = 10

parseInt("10", 8);;

Output = 8

parseInt("0x10");;

Output = 16

parseInt("10", 16);

Output = 16
