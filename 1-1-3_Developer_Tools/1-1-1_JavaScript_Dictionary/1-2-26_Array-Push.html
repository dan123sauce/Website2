Array.Push()


Defination:

* Push adds items to arrays. The push() method changes the length of the array.

* The push() method produces a new length. It changes the array directly without returning anything.

* The push() method adds new items to the end of an array.

* The push() method changes the length of the array.

* The push() method returns the new length.


Syntax - array.push(item1, item2, ..., itemX)

* item1- The item(s) to add to the array.

* item2 - Minimum one item is required.

* itemX - Last Item of the Array.


Example:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

Add a new item to an array.

fruits.push("Kiwi");

-------------------------------------------------------------------------------------------------------------- --------------------------------------------------------------------------------------------------------------

const fruits = ["Banana", "Orange", "Apple", "Mango"];

Add two new items to an array.

fruits.push("Kiwi", "Lemon");

 

 

-------------------------------------------------------------------------------------------------------------- --------------------------------------------------------------------------------------------------------------

const fruits = ["Banana", "Orange", "Apple", "Mango"];

Add three new items to an array.

fruits.push("Kiwi", "Lemon", "Pineapple");
