Using console.log()


Defination:

* For debugging purposes, you can call the console.log() method in the browser to display data.

* The log() method writes (logs) a message to the console..



Syntax:

console.log(message)


Example of Using console.log()

const myObj = {firstname:"John", lastname:"Doe"};

- will print the value of myObj which contains 2 parameters (first and last name)

console.log(myObj);

------------------------------------------------------------------------

const myArr = ["Orange", "Banana", "Mango", "Kiwi"];

- will print the value in the array of myArr

console.log(myArr);

------------------------------------------------------------------------

console.log(5 + 6);

- will print the value 11
