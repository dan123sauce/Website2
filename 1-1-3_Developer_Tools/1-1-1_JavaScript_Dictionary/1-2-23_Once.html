Once


Defination:

* Once lets you run a command once and only once during the execution of an application. Even though we execute or call this function multiple times then also it will have no effect. The original function’s values will only be returned each time it is called. You mostly find these commands in event listeners.

* Sometimes, you must manually poll to see if something stops or if there is an exception.

* It is a complex command to use though, but it is worth the effort.

* Search box suggestions, text-field auto-saves, and eliminating double-button clicks are all use cases for debounce.

* Debounce lets us trigger a function, but only once per use case.


Syntax - _.once(function)

* Parameters: It takes only one argument, i.e., the function that needs to be called only a single time.

* Return value: It returns the original call’s value each time the function is iteratively or repeatedly called.


This is a simple implementation of the debounce function:

var a = 10;

 

 

function add() {

    a += 10;

}

var startFunc = _.once(add);

 

 

startFunc();
