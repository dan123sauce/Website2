HTML DOM Element classList


Defination:

* The classList property returns the CSS classnames of an element.

* The classList property returns a DOMTokenList (A DOMTokenList is a set of space separated tokens.).


Syntax:

element.classList

--- object = A DOMTokenList.

--- object = A list of the class names of an element.


Examples:


const list = element.classList;

- Add a "myStyle" class to an element:

list.add("myStyle");

------------------------------------------------------------------------------------- -------------------------------------------------------------------------------------

const list = element.classList;

- Remove the "myStyle" class from an element:

list.remove("myStyle");

------------------------------------------------------------------------------------- -------------------------------------------------------------------------------------

const list = element.classList;

- Toggle "myStyle" on and off:

list.toggle("myStyle");

------------------------------------------------------------------------------------- -------------------------------------------------------------------------------------

element.classList.add("myStyle", "anotherClass", "thirdClass");
