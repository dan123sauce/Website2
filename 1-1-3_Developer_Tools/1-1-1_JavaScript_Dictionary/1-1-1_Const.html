Const


Defination:

*Declaring a variable using Const is used when the variable should not be changed


Use const when you declare:

* A new Object

* A new Function

* A new Array


Example: Object:

const var = document.querySelector("input"); - takes the value in the input object and stores in the variable called 'var'. const var = document.querySelector(".class-a .class-b"); - takes the value in the object that holds the class-a and class=b and stores in the variable called 'var'

const var = document.getElementsByClassName('btn')[0];

- Get all elements with class='btn' and stores it in 'var'.

const buttons = document.querySelectorAll('button')

- anything with the (element) of button Saves in the variable 'buttons'.

const gameBoard = document.querySelector('#gameboard')

- This will look at entire document with anything with the ID of 'gameboard'.

const squares = document.querySelectorAll('.grid div')

- This will look at entire document with anything with the class of 'grid div' and store in variable 'squares'.
